🌌 第一章：世界观与核心机制
1.1 正在死去的世界
你所在的行星围绕着一颗步入暮年的恒星“提兰”。恒星活动的不稳定导致了：

周期性辐射峰值：游戏内每 3-7个游戏日 随机发生，期间地表暴露将获得“辐射中毒”效果（生命恢复抑制、随机饥饿）。

地磁减弱：导致夜间怪物生成率提升，且可能出现 “辐照僵尸” （生命值更高，死亡时产生小范围毒性爆炸）。

气候紊乱：随机地点可能突发 “酸雨” （腐蚀非石质/金属方块，对玩家造成持续伤害）或 “超级风暴” （强风效果，并可能卷走未固定的物品实体）。

1.3 文明进度：你们唯一的希望
进度通过完成 “文明里程碑” 推进。每一个里程碑都是一项艰巨的工程挑战。进度会影响行星状态恶化的速度，是你们对抗末日的武器。

⏳ 第二章：开局：落地第一个小时的生死时速
目标：建立抗辐射地下前哨，并找到至少两名队友。

分钟 0-15：绝对冷静，收集与挖掘
拒绝伐木：不要花时间砍树！第一个白天极其宝贵。优先徒手收集20个以上原石制作石器。

寻找食物：沿途击杀动物获取生肉，并挖掘至少10个煤炭。这是你初期的生命与光源。

定位洞穴：寻找一个开阔的、向下延伸的洞穴入口。这是你地下基地的蓝图。

分钟 15-45：深入地下，建立基地
向下挖掘：在Y=40至Y=20层建立你的 “一号抗辐射前哨” 。确保有2x2的空间供至少4人容身。

基础设施：

制作 4个熔炉，持续烧制食物和玻璃。

建立 小型农场：利用骨粉催熟找到的种子，种植小麦和土豆。

水源：必须挖出一个2x2的水池，这是生命和未来农业的保障。

通讯：在公屏打出你的坐标和职业意向（例如：[求生] 坐标 (100, 40, -200) 寻找工业工程师与后勤合作！）。

分钟 45-60：规划与分工
职业选择：与最初的队友明确分工。参考第三章。

制定6小时计划：目标统一为——为 工业工程师 凑齐第一套 青铜/铁制 工具与机械的材料。

终极问答
Q：我们团队只有3个人，能通关吗？
A：理论上可以，但每人需精通至少两个职业，且游戏时间需增加2-3倍。建议寻找更多志同道合者。

Q：在“崩溃边缘”状态，我们真的会全部死亡、存档报废吗？
A：不会完全报废。如果方舟未能建成，文明将“失败”。服务器会生成一个 “遗迹世界” ：在数百年后的新星球上，后来的探险者会发现你们文明的废墟，其中可能包含让你们“复活”或传承科技的物品。这是一个悲壮但充满叙事感的结局。

Q：如何防止团队成员偷窃或破坏？
A：① 使用领地插件（如 LuckPerms + GriefPrevention）。② 建立信任后再开放高级权限。③ 核心物资（如聚变燃料）由 后勤 和 工业 双人权限锁保管。

Q：这个整合包对电脑配置要求高吗？
A：由于大量机械与实体，客户端需要8GB以上内存分配，服务器建议 16GB物理内存 起步，并安装性能优化模组。
