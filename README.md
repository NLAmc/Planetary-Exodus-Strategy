## 🌌 项目简介

**《行星迁移计划 / Planetary Exodus》**  
是一个面向 **长期服务器运营** 的  
**硬科幻 · 文明级 Minecraft Java 模组**。

在这个世界中：

- 行星环境正在持续恶化  
- 恒星活动逐渐失控  
- 生态、资源与结构逐渐崩坏  
- 原地生存 = 缓慢灭亡  

玩家的唯一出路是：

> **推动整个文明，完成行星级迁移。**

这是一次 **工程、组织与时间的考验**。

---

## 🪐 核心世界观

你所处的世界，是一颗正在走向终局的行星。

- 母恒星进入高活跃周期  
- 辐射与能量波动持续上升  
- 气候与生态系统开始失稳  

服务器不会突然毁灭世界，  
但**环境会一天天变坏**。

## 🔁 服务器核心循环（长期可运行）

```text
资源开采
   ↓
工业体系建立
   ↓
能源与基础设施
   ↓
轨道与太空工程
   ↓
行星危机加剧
   ↓
文明级决策
   ↓
阶段性迁移
🌞 行星危机系统（全服共享）
状态	描述	世界影响
🟢 STABLE	相对稳定	接近原版体验
🟡 STRAINED	负荷上升	能源效率变化
🟠 DEGRADED	明显恶化	天气异常、作物衰减
🔴 COLLAPSING	崩溃边缘	辐射、结构损坏、死亡惩罚

🚀 迁移工程（不是换地图）
行星迁移是一项真正的文明工程：

建造轨道设施

稳定近地空间

转移工业与人口

冻结部分地表

激活新宜居区域 / 星系

🛰 火箭与太空系统
火箭是 服务器实体，不是 GUI 按钮。

典型飞行阶段：

发射（燃料消耗 + 风险）

轨道插入（可能失败）

推进与航行（时间推进）

抵达轨道设施 / 新区域

👥 玩家分工
自然形成的角色：

🛠 工业工程师

🚀 航天工程师

🛰 轨道设施维护

🔬 科研人员

📦 后勤协调

☠️ 失败不是删档

「衰亡纪元 / Decline Era」模式
